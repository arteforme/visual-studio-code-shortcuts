Explorer:  **`shift-cmd-E`**

# Search
* Global Find:  **`shift-cmd-F`**
* Global Find & Replace:  **`shift-cmd-H`**

Source Control:  `ctrl-shift-G`

# Editor
* Split:  **`cmd-k`**, **`↑` | `→` | `↓` | `←`**
* Move to Next Group:  **`ctrl-cmd-→`**

# Go TO
* Line:  **`ctrl-g`**
* Last Edit Location:  **`cmd-k, cmd-q`**
